# IONOS Cloud Foundational Exam: Strategy & Details

## Exam Overview
- **Structure**: 40 questions
- **Format**: Multiple-choice (4 options per question), Use Case questions, and True/False questions.
- **Time Limit**: 1 Hour
- **Logistics**: The exam is proctored and will be held as an online meeting.

## Key Topics to Review (Common Pitfalls)
Based on feedback from previous test-takers, ensure you are solid on the following specific facts:

1. **Monitoring Service Metrics**: 
   - The default push interval is **1 minute**.
2. **SSH Keys**: 
   - The maximum number of SSH IDs you can configure is **100**.
3. **Networking (IPv6)**:
   - An IPv6 address size is **128 bit**.
4. **Cloud Concepts (Scalability vs. Elasticity)**:
   - *Scalability*: Refers to **resource management** (adding/removing resources to handle load).
   - *Elasticity*: Refers to **automation based on demand** (automatically scaling up or down dynamically).
5. **IONOS Value Proposition**:
   - For private cloud TCO (Total Cost of Ownership), IONOS offers **40–60%** (or up to **70%**) savings compared to on-premise solutions.

## Preparation Strategy
- **Step 1:** Review the Unit Summary Guides to solidify core concepts of Cloud Basics, Core Services, and Management & Operations.
- **Step 2:** Ensure you can differentiate between similar concepts (like Scalability vs. Elasticity) as use-case questions will test this.
- **Step 3:** Memorize the strict numerical limits and default values (e.g., 100 SSH keys, 1 minute push interval).
- **Step 4:** Take the comprehensive practice quizzes for each unit, aiming for >85% accuracy to ensure readiness for the 40-question exam format.
