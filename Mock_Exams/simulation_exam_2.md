## Part 2: True / False (Fact Checking)

**16. True or False:** "Elasticity" refers to the long-term, manual process of adding larger servers to handle business growth over years.
A. True
B. False

**17. True or False:** Cross Connect allows you to span a private, free LAN connection between two VDCs, provided they are in the same region and contract.
A. True
B. False

**18. True or False:** A Cost Alert set to $500 will automatically shut down all running VMs once the contract’s net invoice hits $500.
A. True
B. False

**19. True or False:** In-Memory DB is a RAM-based caching solution that is Redis-compatible.
A. True
B. False

**20. True or False:** IONOS Cloud operates exclusively under European jurisdiction, making it immune to the US CLOUD Act.
A. True
B. False

**21. True or False:** Providing power, cooling, and physical security to the data centers is the responsibility of the customer under the Shared Responsibility Model.
A. True
B. False

**22. True or False:** If a user belongs to a group, they automatically inherit all privileges assigned to that group.
A. True
B. False

**23. True or False:** The Application Load Balancer (ALB) operates at Layer 4 of the OSI model and cannot route traffic based on URL paths.
A. True
B. False

**24. True or False:** You cannot add RAM or CPU capacity to a running Dedicated Core server without restarting it, due to physical hardware limitations.
A. True
B. False (Note: Live Vertical Scaling allows adding resources without downtime on supported OS).

**25. True or False:** Cloud Savings Plans are "Instance-based," meaning if you commit to a specific VM size in Frankfurt, you cannot apply that discount to a different VM size in Berlin.
A. True
B. False

## Part 3: Use Cases (Scenario Analysis)

**26. Scenario:** A financial institution runs a high-frequency trading application. They require guaranteed compute performance without any "noisy neighbor" interference or "steal time" from other tenants. 
**Action:** Which compute option should they provision?
A. vCPU Servers
B. Dedicated Core Servers
C. Cubes
D. VM Auto Scaling cluster of Cubes

**27. Scenario:** A hospital stores millions of archived X-ray images. The images are accessed rarely, but must be retained for 10 years due to compliance. They need the cheapest possible storage option that supports HTTP access.
**Action:** Which storage solution is best?
A. SSD Premium Block Storage
B. Network File Storage (NFS)
C. Block Storage (HDD)
D. IONOS Cloud Object Storage

**28. Scenario:** A web development agency runs 50 separate e-commerce websites. They want a single load balancer that can look at the incoming HTTP request and route `shopA.com/products` to Server Group 1, and `shopB.com/products` to Server Group 2.
**Action:** Which load balancer is required?
A. Managed Application Load Balancer
B. Managed Network Load Balancer
C. VPN Gateway
D. Managed NAT Gateway

**29. Scenario:** An administrator receives an email stating, "Your net invoice amount has reached 75% of your budget." Two days later, costs increase to 90% of the budget. 
**Action:** How many additional emails will the administrator receive for crossing the 90% mark, assuming they only set one alert at 75%?
A. Zero. Alerts only trigger once per defined threshold.
B. One. The system automatically shifts the threshold up by 15%.
C. Two. One at 80% and one at 90%.
D. It will send emails every hour until the administrator logs in.

**30. Scenario:** A malicious insider is suspected of logging in at 3:00 AM and deleting a production database. The security team needs to verify exactly which user account initiated the delete request via the API.
**Action:** Which service should the security team query to find this information?
A. Flow Logs
B. Monitoring Service Grafana Dashboard
C. Activity Logs
D. Network Security Group rules

**31. Scenario:** A startup is evaluating moving their on-premise infrastructure to IONOS Private Cloud. They need to present a cost-benefit analysis to their investors.
